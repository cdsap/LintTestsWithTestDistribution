Retaining a `Context` (or another type that holds a strong reference to a `Context`) as either the return type of `calculation` or as a key input to `retain` will leak the Context and prevent its memory from being properly reclaimed. If caching is necessary, consider remembering offending values instead of retaining them. !!! Tip This lint check has an associated quickfix available in the IDE.
